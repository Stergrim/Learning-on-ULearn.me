# Прерывание транзакции

Прочитай [про команду TRUNCATE](https://postgrespro.ru/docs/postgresql/13/sql-truncate). Обрати внимание на параметры.

Находясь в транзакции, ты случайно очистил таблицу `ticket_flights` и **все таблицы, ссылающиеся по внешнему ключу на нее.** Нужно отменить эти изменения.

**Задание**
1. В блоке транзакции очисти таблицы.
2. Отмени изменения транзакции.

**Все тесты пройдены, задача сдана:**
```pgsql
Begin;
Truncate Table ticket_flights Cascade;
Rollback;
```
