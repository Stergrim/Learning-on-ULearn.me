 > Почему они назвали меня зайкой? Точно! Меня зовут Зайцев А... Не могу вспомнить имя, но номер моего паспорта начинается с трех семерок.
 > Нужно попробовать найти билеты оформленные на меня. Ага, вот и таблица `tickets`.
 
| **Столбец**    | **Тип**     | **Модификаторы** | **Описание**                |
|:---------------|:------------|:-----------------|:----------------------------|
| ticket_no      | char(13)    | not null         | Номер билета                |
| book_ref       | char(6)     | not null         | Номер бронирования          |
| passenger_id   | varchar(20) | not null         | "Серия Номер" паспорта      |
| passenger_name | text        | not null         | "Имя Фамилия" пассажира     |
| contact_data   | jsonb       |                  | Контактные данные пассажира |

**Задание**

Выбери из таблицы tickets первые пять билетов принадлежащие пассажиру с фамилией `ZAYCEV`, а имя с буквы `A`. Номер паспорта должен начинаться с `777`.
Получите серию и номер паспорта, **только имя** и контактные данные.
Сортировка должны быть по возрастанию серии и номера паспорта.

**Все тесты пройдены, задача сдана**
```pgsql
Select
passenger_id,
Left(passenger_name, STRPOS(passenger_name, ' ') - 1) as first_name,
contact_data
From tickets
Where passenger_name Like 'A% ZAYCEV' And passenger_id Like '____ 777___'
Order by passenger_id
Limit 5
```
