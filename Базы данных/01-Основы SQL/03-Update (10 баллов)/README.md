# Update

Вы решили подстраховаться и изменить добавленные строки таким образом, чтобы они выглядели правдивее.

Таблица user_privileges

| **Столбец**   | **Тип**      | **Модификаторы** | **Описание**                 |
|:--------------|:-------------|:-----------------|:-----------------------------|
| **user_id**   | int          | not null         | Идентификатор пользователя   |
| **privilege** | varchar(20)  | not null         | Название привилегии          |
| description   | varchar(255) |                  | Описание                     |
| expires       | timestamp    | not null         | Время, до которого действует |

**Задание**

Обновите информацию для привилегий своего пользователя (идентификатор: `1618033`). Увеличьте дату действия привилегий на `1 год`. Задайте описание: `Теперь он послушный зайка`.

**Все тесты пройдены, задача сдана:**
```pgsql
Update user_privileges
Set
expires = expires + Interval '1 year',
description = 'Теперь он послушный зайка'
Where user_id = 1618033
```
