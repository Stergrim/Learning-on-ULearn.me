# Блокировки на уровне таблицы

Посмотри в [документации](https://postgrespro.ru/docs/postgresql/13/sql-lock) как использовать блокировки на уровне таблиц и приступай к заданию.

**Задание**
1. Заблокируй таблицу `seats`, чтобы никто не мог ее использовать кроме текущий транзакции.
2. Заблокируй таблицу `boarding_passes`, чтобы другим транзакциям из нее можно было только читать.
3. Посмотри список блокировок с помощью представления pg_locks
4. Проверьте и убедитесь, что при попытке считать данные из seats запрос будет ожидать снятия разблокировки (отправлять в задание не нужно).

**Все тесты пройдены, задача сдана:**
```pgsql
Begin;

Lock Table seats In Access Exclusive Mode;
Lock Table boarding_passes In Exclusive Mode;

Select locktype, relation::regclass::varchar, Mode From pg_locks;

Commit;
```
