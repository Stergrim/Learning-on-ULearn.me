# Итоговая практика: Шаблонизатор Jinja

Продолжайте в том же [проекте](task.zip).

В этом задании вам предстоит создать одну из страниц при помощи втроенного шаблонизатора Jinja.

В проекте уже находится созданная заготовка необходимого файла: djangotask/templates/vacancy.html, вам необходимо наполнить её.

На вход поступают следующие параметры: * vacancy: Экземпляр модели Vacancy * skills: Список названий навыков, необходимых для вакансии * responses: QuerySet откликов на эту вакансию.

Вам необходимо:
- Описать вакансию
- Перечислить требуемые навыки
- Перечислить отклики к данной вакансии

Перечисление требуемых навыков и откликов необходимо сделать при помощи неупорядоченных списков.

**Формат вывода**

В результате полученная страница должна выглядеть следующим образом:

```
Разработчик Django
Зарплата: 100000
Город: Москва
Требуемые навыки:

Python
Django
REST

Отклики:

Иван Иванов: У меня опыт работы с Django 5 лет
Юрий Перов: Кодил на C#, Python, Scala, в превосходстве знаю SQL
Дмитрий Романов: Возьмите меня на работу пожалуйста
```

Все тесты пройдены, задача сдана:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>ans</title>
</head>
<body>
<div id="answer">
    <p>{{vacancy.name}}</p>
    <p>Зарплата: {{vacancy.salary}}</p>
    <p>Город: {{vacancy.area_name}}</p>
    <p>Требуемые навыки:</p>
    <p></p>
    {% for skill in skills %}
        <p>{{skill}}</p>
    {%endfor%}
    <p></p>
    <p>Отклики:</p>
    <ul>
        {% for response in responses %}
        <li>{{response.user.first_name}} {{response.user.last_name}}: {{response.message}}</li>
        {% endfor %}
    </ul>
</div>
</body>
</html>
```
