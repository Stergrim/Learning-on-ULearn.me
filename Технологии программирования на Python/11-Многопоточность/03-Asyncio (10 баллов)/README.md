# Asyncio

К Васе на работу собирается приехать какая-то важная и срочная делегация с какой-то важной и срочной проверкой. И для этой важной и срочной делегации нужно собрать имена всех сотрудников. А имена всех сотрудников можно получить только из их досье. Так как сотрудников очень много (десятки тысяч человек), то собирать ручками имена стало как-то затратно по времени.

Сеньор Дон Эладио сказал, что у него в далёкой-далёкой папке на диске J: лежат всякие функции, с помощью которых можно считывать из досье сотрудников их имена.

Вася написал великолепную программу, которая проходится по списку из досье сотрудников (`filenames`) и с помощью функции `read_file_sync` собирает нужные данные. Потом складывает все получившиеся имена в одну строку и выписывает её.

Но вот незадача, его программа оказалась слишком медленной. Имена просто не успеют выгрузиться до приезда делегации! Однако Вася не отчаялся. Он нашёл под пылью ноликов и единиц функцию `read_file_async`, которая, возможно, поможет ему ускорить процесс! Помогите Васе оптимизировать его код с помощью новонайденной асинхронной функции. Возможно, вам для этого потрубется [asyncio](https://docs.python.org/3/library/asyncio.html).

|                         |               |
|:------------------------|:--------------|
| **Ограничение времени** | **12 секунд** |
| **Ограничение памяти**  | **2048 Мб**   |


Все тесты пройдены, задача сдана:
```py
import asyncio

async def main(filenames):
    tasks = [read_file_async(filename) for filename in filenames]
    names = await asyncio.gather(*tasks)
    return ' '.join(names)
```
