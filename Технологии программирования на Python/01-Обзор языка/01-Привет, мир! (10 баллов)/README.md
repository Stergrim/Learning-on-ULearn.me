# Привет, мир!

Вася решил начать изучение языка Python. Первым делом он по традиции приступил к написанию скрипта, выводящего на экран фразу "Hello, world!". Чтобы не терять времени, Вася решил заодно и по-подробнее изучить функцию print.

Потренируйтесь с Васей в использовании данной функции языка Python!

Все тесты пройдены, задача сдана:
```py
# Необходимо поменять аргумент функции, чтобы на экран выводилась фраза "Hello, world!"
print("Hello, world!")

print("H", "e", "l", "l", "o", ",", " ", "w", "o", "r", "l", "d", "!", sep='')

print("Hello,", end=' ')
print("world", end='!')
```
