## [Технологии программирования на Python](https://ulearn.me/course/python-rtf-learning/Struktura_kursa_yazyka_Python_769b0e67-6156-4fba-99c6-de53ac08e293)

Курс рассчитан на людей имеющий базовые знания в программировании. Начинается курс с основ языка: синтаксиса, конструкций и парадигм программирования. Далее рассматриваются популярные области применения языка python, такие как анализ данных и веб-фреймворк Django.
