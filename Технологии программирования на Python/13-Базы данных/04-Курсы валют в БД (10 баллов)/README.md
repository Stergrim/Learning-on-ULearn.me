# Курсы валют в БД

Васе пришла новая задача. Для того, чтобы дальше можно было расширять функциональность его приложения и масштабировать его, необходимо было перенести хранение данных из [CSV-файла](vacancies.zip) в базу данных. Для начала, Вася взял наиболее простую реализацию в качестве БД - это SQLite.

Как обычно, Вася и Коля разделили большую задачу на набор небольших задач и приступили к делу. Первым этапом надо было перенести курсы валют, которые они извлекли с сайта ЦБ РФ в базу данных.

Вася написал ТЗ на первую часть работ:
1. Необходимо перенести хранение таблицы с курсами валют в БД SQLite (В случае использования Pandas, удобно использовать функцию df.to_sql()).
2. Структура таблицы хранения не должна измениться от переноса.
3. Необходимо соблюдать, чтобы тип для хранения данных в SQLite совпадал со значением.

На вход подаются: название базы данных, название csv-файла с сохраненными курсами валют и название таблицы, в которую нужно перенести данные.

**Формат ввода**

```
example_database
example_currency.csv
answer_table
```

**Формат итоговой таблицы**

| **date**    | **BYR**       | **USD**     | **EUR**     | **KZT**      | **UAH**     | **AZN**     | **KGS**      | **UZS**       |
|:------------|:--------------|:------------|:------------|:-------------|:------------|:------------|:-------------|:--------------|
| **2010-08** | **0.0101434** | **30.1869** | **39.4694** | **0.204352** | **3.82316** | **37.5646** | **0.647093** | **0.0188016** |
| **2010-09** | **0.0102446** | **30.8669** | **39.0127** | **0.209487** | **3.91364** | **38.4108** | **0.66167**  | **0.0191362** |

|                         |               |
|:------------------------|:--------------|
| **Ограничение времени** | **3 секунды** |
| **Ограничение памяти**  | **512.0 Мб**  |


Все тесты пройдены, задача сдана:
```py
import pandas as pd
import sqlite3

database_name = input()
csv_file = input()
table_name = input()

df = pd.read_csv(csv_file, encoding="utf-8-sig")
conn = sqlite3.connect(database_name)
df.to_sql(table_name, conn, index=False)
conn.close()
```
