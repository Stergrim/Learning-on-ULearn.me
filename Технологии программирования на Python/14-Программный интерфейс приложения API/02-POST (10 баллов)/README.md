# POST

Теперь Васе предстоит через API создать новую вакансию! На вход должны поступать следующие данные:
1. name - Название вакансии.
2. salary - Зарплата.
3. area_name - Место, в котором размещается вакансия.

Программа должна отправлять эти данные на сервер. Адрес для POST-запроса: http://127.0.0.1:8000/vacancies

**Формат ввода**

```
Программист
10000
Уфа
```

**Формат вывода**

Программа ничего не должна возвращать.

|                         |              |
|:------------------------|:-------------|
| **Ограничение времени** | **5 секунд** |
| **Ограничение памяти**  | **256 Мб**   |


Все тесты пройдены, задача сдана:
```py
import requests

BASE_URL = 'http://127.0.0.1:8000/vacancies'
name = input()
salary = input()
area_name = input()
post_query = {
    "name": name,
    "salary": salary,
    "area_name": area_name
}

response = requests.post(BASE_URL, json=post_query)
```
