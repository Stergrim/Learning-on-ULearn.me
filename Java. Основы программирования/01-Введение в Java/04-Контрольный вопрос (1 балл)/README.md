# Контрольный вопрос

1. Выберите верные утверждения про формат JAR. Может быть выбрано несколько пунктов. (1 из 1 балла)
   * ✅ **В JAR-архиве может быть специальный файл с метаданными — META-INF/MANIFEST.MF.** (Правильно!)
   * ❌ **В JAR-файл разрешено запаковывать только class-файлы.**
   * ❌ **Формат JAR основан на формате RAR.**
   * ✅ **Формат JAR основан на формате ZIP.** (Правильно!)
