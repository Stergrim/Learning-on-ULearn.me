# Контрольный вопрос

Выберите верные утверждения про модификатор static.

1. Тест — Выберите один или несколько вариантов из списка (1 из 1 балла)
   * ❌ **Поле класса, объявленное с модификатором static, — это константа, т.е. значение не может меняться.**
   * ✅ **К полям и методам, имеющим модификатор static, можно обращаться, не создавая экземпляр содержащего их класса.** (Правильно!)
   * ❌ **Класс, объявленный с модификатором static, может иметь только статические методы.**
   * ✅ **Вложенный класс, объявленный с модификатором static, теряет возможность обращаться к нестатическим членам внешнего класса.** (Правильно!)
