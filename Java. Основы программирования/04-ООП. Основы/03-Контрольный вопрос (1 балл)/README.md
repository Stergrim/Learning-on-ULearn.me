# Контрольный вопрос

Допустим, вы пишете класс Example в пакете com.example.java. Классы из каких пакетов вы можете использовать в классе Example по их коротким именам без явного import'а?

1. Тест — Выберите один или несколько вариантов из списка (1 из 1 балла)
   * ❌ **все подпакеты org.stepic.java.example**
   * ✅ **com.example.java** (Правильно!)
   * ❌ **java.util**
   * ✅ **java.lang** (Правильно!)
   * ❌ **пакет по умолчанию**
