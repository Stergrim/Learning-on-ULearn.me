# Контрольный вопрос

Какие модификаторы доступа применимы к классу верхнего уровня (т.е. не вложенному в другой класс)?

Выберите верные утверждения

1. Тест — Выберите один или несколько вариантов из списка (1 из 1 балла)
   * ✅ **отсутствие модификатора (пакетный доступ)** (Правильно!)
   * ✅ **public** (Правильно!)
   * ❌ **private**
   * ❌ **protected**
