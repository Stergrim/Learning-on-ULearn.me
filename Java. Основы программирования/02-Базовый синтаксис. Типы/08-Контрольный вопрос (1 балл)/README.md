# Контрольный вопрос

Укажите преобразования типов, которые компилятор делает автоматически. Другими словами, для каких преобразований не требуется явно писать в программе оператор приведения типа?

1. Выберите один или несколько вариантов из списка. (1 из 1 балла)
   * ✅ **char -> int** (Правильно!)
   * ✅ **long -> float** (Правильно!)
   * ❌ **float -> long**
   * ❌ **String -> int**
   * ❌ **int -> boolean**
   * ❌ **byte -> char**
   * ✅ **char -> Character** (Правильно!)
   * ✅ **int -> long** (Правильно!)
