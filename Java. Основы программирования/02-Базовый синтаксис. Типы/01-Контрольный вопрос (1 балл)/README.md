# Контрольный вопрос

Укажите пары эквивалентных булевских операторов.

Операторы эквивалентны, если их [таблицы истинности](https://ru.wikipedia.org/wiki/%D0%A2%D0%B0%D0%B1%D0%BB%D0%B8%D1%86%D0%B0_%D0%B8%D1%81%D1%82%D0%B8%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D0%B8) совпадают, т.е. для любой пары аргументов оба оператора дают один и тот же результат.

Если ответ не очевиден, то составьте таблицы истинности для всех упомянутых операторов и сравните их.

1. Выберите один или несколько вариантов из списка. (1 из 1 балла)
   * ✅ **| и ||** (Правильно!)
   * ❌ **& и ==**
   * ❌ **| и !=**
   * ✅ **^ и !=** (Правильно!)
