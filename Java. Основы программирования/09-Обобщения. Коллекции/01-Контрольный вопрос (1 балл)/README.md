# Контрольный вопрос

Предположим, у нас есть параметризованный класс:

```java
public class Example<X> {
  ...
}
```

Что можно подставлять в качестве значения параметра X при использовании этого класса в программе?

1. Выберите один или несколько вариантов из списка. (1 из 1 балла)
   * ❌ **имя любого примитивного типа (например, int)**
   * ✅ **имя любого перечисления (например, DayOfWeek)** (Правильно!)
   * ❌ **ссылку на метод (например, Object::toString)**
   * ✅ **значение X можно не указывать, т.е. использовать класс Example как обычный непараметризованный** (Правильно!)
   * ✅ **символ "?" или более сложное выражение с ключевыми словами extends и super** (Правильно!)
   * ✅ **имя любого интерфейса (например, CharSequence)** (Правильно!)
   * ✅ **имя любого класса (например, Object)** (Правильно!)
   * ❌ **значение примитивного типа (например, 42)**
