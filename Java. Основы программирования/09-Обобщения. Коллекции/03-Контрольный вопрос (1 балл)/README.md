# Контрольный вопрос

Чем коллекции отличаются от массивов?

1. Выберите один или несколько вариантов из списка. (1 из 1 балла)
   * ✅ **Коллекции могут содержать только объекты, а массивы могут содержать и примитивы.** (Правильно!)
   * ✅ **Массивы встроены в язык, а коллекции — обычные классы стандартной библиотеки.** (Правильно!)
   * ✅ **Коллекции могут динамически менять размер, а размер массива фиксируется при создании.** (Правильно!)
   * ✅ **Коллекции можно сравнивать по содержимому обычным методом equals(), а для сравнения массивов используется внешний утилитный метод Arrays.equals().** (Правильно!)
   * ❌ **Массивы можно использовать в цикле foreach, а коллекции нельзя.**
