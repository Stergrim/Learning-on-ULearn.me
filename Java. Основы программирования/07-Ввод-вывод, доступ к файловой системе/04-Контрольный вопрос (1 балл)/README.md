# Контрольный вопрос

Найдите среди приведенных путей два эквивалентных, то есть указывающих на один и тот же файл.

1. Тест — Выберите один или несколько вариантов из списка (1 из 1 балла)
   * ✅ **.\a\b\..\b\c\.\file.txt** (Правильно!)
   * ✅ **a\b\c\file.txt** (Правильно!)
   * ❌ **a\b\..\file.txt**
   * ❌ **a\..\b\c\file.txt**
   * ❌ **a\.\b\..\c\.\file.txt**
