# Контрольный вопрос

Выберите верные утверждения про интерфейсы и абстрактные классы.

1. Тест — Выберите один или несколько вариантов из списка (1 из 1 балла)
   * ❌ **Интерфейсы отличаются от абстрактных классов тем, что не могут содержать реализаций методов.**
   * ✅ **Интерфейсы отличаются от абстрактных классов тем, что могут содержать только публичные методы и публичные статические финальные поля.** (Правильно!)
   * ✅ **Класс может наследоваться только от одного класса, но может реализовывать сколько угодно интерфейсов.** (Правильно!)
   * ❌ **Ключевое слово "interface" — это сокращенная запись для "abstract class".**
