# Контрольный вопрос

Какие стримы есть в стандартной библиотеке?

1. Выберите один или несколько вариантов из списка. (1 из 1 балла)
   * ❌ **CharStream**
   * ❌ **ByteStream**
   * ✅ **Stream** (Правильно!)
   * ✅ **DoubleStream** (Правильно!)
   * ✅ **LongStream** (Правильно!)
   * ❌ **StringStream**
   * ❌ **FloatStream**
   * ✅ **IntStream** (Правильно!)
