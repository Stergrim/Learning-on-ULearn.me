# Контрольный вопрос

Выберите верные утверждения.

1. Выберите один или несколько вариантов из списка. (1 из 1 балла)
   * ✅ **Стрим позволяет применять любое количество промежуточных операций. Все они запустятся только тогда, когда на стриме будет вызвана терминальная операция.** (Правильно!)
   * ❌ **Элементами стрима могут быть только объекты, стримы примитивов не разрешены.**
   * ✅ **Стримы "одноразовые", т.е. после вызова терминальной операции стрим больше не пригоден к использованию.** (Правильно!)
   * ❌ **Стрим надо обязательно всегда закрывать, т.е. вызывать метод его close() или использовать в блоке try с ресурсами.**
   * ❌ **Применение трансформаций к элементам стрима изменяет содержимое источника, откуда стрим берет элементы (например, коллекции или массива).**
