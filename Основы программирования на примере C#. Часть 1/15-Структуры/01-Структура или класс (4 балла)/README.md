# Структура или класс?

Изучите следующий код:

```cs
X[] xs = new X[10];
xs[0].A = 1;
```

1. Если этот код компилируется и работает, то чем может быть X? (1 из 1 балла)
   * ✅ **X может быть структурой** (Правильно!)
   * ❌ **X может быть классом**
   * ❌ **X не может быть ни структурой, ни классом**


Изучите следующий код:

