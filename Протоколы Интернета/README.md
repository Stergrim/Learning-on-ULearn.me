## [Протоколы Интернета](https://ulearn.me/course/internet/51413213-790b-44ba-a178-9d8fae81e116)

Целью данного курса является знакомство студентов с основами работы компьютерных сетей и протоколов Интернета.

После видеолекций предлагаются тесты, которые помогут лучше понять и запомнить материал, поэтому не пропускайте их! Старайтесь решать тесты сразу после просмотра записей и снова через некоторое время (1 час и более). Для решения некоторых тестов недостаточно озвученных в видео данных, иногда придётся искать информацию или делать выводы самостоятельно. Не пугайтесь этого! А также, будьте внимательны при чтении задания.
