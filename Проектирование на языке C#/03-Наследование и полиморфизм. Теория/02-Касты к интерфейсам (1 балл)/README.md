# Касты к интерфейсам

```cs
interface A { }
interface B : A { }

class X : A {}
class Y : X, B {}
class Z : X {}
```

1. Какие из этих следующих операторов не выбросят исключение? (1 из 1 балла)
   * ✅ **X as B** (Правильно!)
   * ✅ **(A) Y** (Правильно!)
   * ✅ **Y as B** (Правильно!)
   * ❌ **(B) Z**
   * ✅ **(A) X** (Правильно!)
