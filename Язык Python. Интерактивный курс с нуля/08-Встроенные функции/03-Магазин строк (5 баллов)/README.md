Вася выбирает между двумя строками: *s* и *t*. Обе они очень ему нравятся, но он хочет купить только одну из них.

Тариф в магазине строк очень прост: цена строки равна ее длине.

Вася хочет купить ту, что стоит **дороже**. Сообщите стоимость этой строки.

**Входные данные**

В первой строке задана строка *s* из латинских букв *(1≤∣s∣≤10)*.

Во второй строке задана строка *t* из латинских букв *(1≤∣t∣≤10)*.

**Выходные данные**

Выведите единственное целое число — ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| first              |
| second             |

| **Выходные данные** |
|---------------------|
| 6                   |


Все тесты пройдены, задача сдана:
```py
s = input()
t = input()
print(max(len(s), len(t)))
```
