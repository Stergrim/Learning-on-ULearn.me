Вам нужно срочно занять *a* долларов в банке.

Первый банк предлагает вам получить эти *a* долларов, но взамен вы должны будете ему вернуть *3⋅a* долларов через год.

Второй банк предлагает вам получить *a* долларов, но взамен нужно будет вернуть *2⋅a+100* долларов через год.

Выведите, какое наименьшее число долларов нужно будет вернуть одному из двух банков. 

**Входные данные**

В единственной строке вводится целое число *a* *(1≤a≤1000)*.

**Выходные данные**

Выведите единственное целое число — ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 20                 |

| **Выходные данные** |
|---------------------|
| 60                  |

| **Входные данные** |
|--------------------|
| 1000               |

| **Выходные данные** |
|---------------------|
| 2100                |


Все тесты пройдены, задача сдана:
```py
a = int(input())
s1 = 3*a
s2 = 2*a + 100
print(min(s1, s2))
```
