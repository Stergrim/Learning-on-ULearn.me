У Васи есть задача, которую можно решить на компьютере Коли и Вовы.

Колин компьютер *a* секнунд включается и за *b* секунд сможет решить Васину задачу. Вовин компьютер *c* секунд включается и за *d* секунд сможет решить Васину задачу.

До обоих мальчиков идти ровно одну минуту. Скажите, через какое наименьшее время в *секундах* задача будет решена. Любыми другими замедляющими факторами пренебрегайте. 

**Входные данные**

В четырех строках заданы целые числа *a,b,c,d* *(1≤a,b,c,d≤1000)*.

**Выходные данные**

Выведите единственное целое число — ответ на задачу в секундах.

**Пример**

| **Входные данные** |
|--------------------|
| 60                 |
| 120                |
| 10                 |
| 30                 |

| **Выходные данные** |
|---------------------|
| 100                 |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
c = int(input())
d = int(input())
print(min(a + b, c + d) + 60)
```
