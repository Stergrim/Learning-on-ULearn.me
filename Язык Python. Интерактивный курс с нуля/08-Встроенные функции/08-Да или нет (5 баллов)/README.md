В разных задачах, в которых требуется вывести «YES» или «NO» иногда нужно выводить только заглавные буквы, а иногда только маленькие.

Назовем задачей первого типа задачу, в которой ответ нужно вывести маленькими буквами. Задача второго типа — это задача, в которой ответ нужно вывести большими буквами.

Вам над тип задачи и ответ на нее — строка «YES» или «NO» в случайных регистрах. Преоборазуйте ответ на эту задачу под необходимый формат, в зависимости от типа задачи. 

**Входные данные**

В первой строке вводится целое число *1* или *2* — тип задачи.

Во второй строке вводится ответ на задачу.

**Выходные данные**

Выведите единственную строку — правильный ответ на задачу, то есть ответ, преобразованный только в маленькие или только в заглавные буквы.

**Пример**

| **Входные данные** |
|--------------------|
| 1                  |
| YeS                |

| **Выходные данные** |
|---------------------|
| yes                 |

| **Входные данные** |
|--------------------|
| 2                  |
| nO                 |

| **Выходные данные** |
|---------------------|
| NO                  |


Все тесты пройдены, задача сдана:
```py
t = int(input())
s = input()
if t == 1:
    print(s.lower())
else:
    print(s.upper())
```
