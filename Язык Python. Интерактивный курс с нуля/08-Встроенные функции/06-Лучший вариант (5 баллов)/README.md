У Коли раз в *a* дней лекции по матанализу и раз в *b* дней лекции по дискретке. При этом существует день, когда у него обе лекции проводятся одновременно.

Ему хочется составить простое расписание, где он раз *c* дней посещает университет, а в остальные дни он ходит на курсы по английскому. Так как курсы увлекательны, он хочет выбрать максимально возможное *c*.

Друзья подсказали Коле, что в качестве *c* тогда необходимо выбрать наибольший общий делитель *a* и *b*. Помогите ему его посчитать!

**Входные данные**

В двух строках вводятся два целых числа: *a* и *b* *(1≤a,b≤1000)*.

**Выходные данные**

Выведите единственное целое число — ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 4                  |
| 6                  |

| **Выходные данные** |
|---------------------|
| 2                   |


Все тесты пройдены, задача сдана:
```py
import math

a = int(input())
b = int(input())
print(math.gcd(a, b))
```
