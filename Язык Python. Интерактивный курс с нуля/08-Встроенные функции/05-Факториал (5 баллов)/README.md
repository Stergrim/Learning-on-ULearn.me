# Факториал

<p align="center">
<b>Факториал</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Дано число *n*. Посчитайте *n!*.

**Входные данные**

В единственной строке вводится число *n* *(1≤n≤20)*.

**Выходные данные**

Выведите единственное целое число — *n!*.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |

| **Выходные данные** |
|---------------------|
| 6                   |


Все тесты пройдены, задача сдана:
```py
import math

n = int(input())
print(math.factorial(n))
```
