## [Язык Python: интерактивный курс с нуля](https://ulearn.me/Course/python/Yazyk_Python_cfe94d1b-80ef-4256-8f7c-303c2c49c24b)

Длительность: ~40 ак.часов.

Практический курс по основам программирования на Python. Подходит для новичков и тех, кто хочет систематизировать знания. Включает теорию, интерактивные упражнения и автоматическую проверку задач. Основные темы: синтаксис Python, типы данных, функции, ООП, работа с файлами и модулями. Доступен бесплатно и предусмотрен учительский доступ для создания групп.
