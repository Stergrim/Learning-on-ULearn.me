# Верное выражение

<p align="center">
<b>Верное выражение</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Даны числа *a1,a2,…,an*. Требуется вывести без пробелов выражение, не меняя порядка символов, вида: *a1+a2+…+an=S*, где *S* — посчитанная сумма всех элементов.

Посмотрите на примеры для прояснения требуемого вывода.

**Входные данные**

В единственной строке вводятся числа *a1,a2,…,an* *(1≤n≤10,1≤a<sub>i</sub><10)*.

**Выходные данные**

Выведите требуемое выражение.

**Пример**

| **Входные данные** |
|--------------------|
| 1 2 3              |

| **Выходные данные** |
|---------------------|
| 1+2+3=6             |


Все тесты пройдены, задача сдана:
```py
a = list(map(int, input().split()))
print(*a, sep='+', end='=' + str(sum(a)))
```
