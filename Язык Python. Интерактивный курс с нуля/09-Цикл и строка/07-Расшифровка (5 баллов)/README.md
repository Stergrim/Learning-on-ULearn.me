# Расшифровка

<p align="center">
<b>Расшифровка</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

У Коли есть сообщение *s* и число *r*. Он ищет в сообщении *s* секретное послание.

Секретное послание в следующем: если количество букв «s» среди первой, второй, ..., *r*-й букв — нечетное число, то ему передали «yes», а иначе ему передали «no».

Сообщите, что Коле передали.

**Входные данные**

В первой строке вводится сообщение *s*, состоящая из строчных латинских букв. Длина этой фразы не превосходит *20*.

Во второй строке вводится целое число *r*, *1≤r≤∣s∣*.

**Выходные данные**

Выведите сообщение, которое передали Коле.

**Пример**

| **Входные данные** |
|--------------------|
| sss                |
| 2                  |

| **Выходные данные** |
|---------------------|
| no                  |


Все тесты пройдены, задача сдана:
```py
s = input()
r = int(input())
if s.count('s', 0, r) % 2 != 0:
    print("yes")
else:
    print("no")
```
