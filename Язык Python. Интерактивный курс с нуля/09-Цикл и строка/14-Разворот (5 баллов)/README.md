# Разворот

<p align="center">
<b>Разворот</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам дана строка *s*. Разверните ее.

**Входные данные**

В единственной строке вводится строка *s*, состоящая из латинских букв *(1≤∣s∣≤50)*.

**Выходные данные**

Выведите развернутую строку *s*.

**Пример**

| **Входные данные** |
|--------------------|
| abcde              |

| **Выходные данные** |
|---------------------|
| edcba               |


Все тесты пройдены, задача сдана:
```py
s = input()
print(s[::-1])
```
