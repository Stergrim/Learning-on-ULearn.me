Василий Петрович попросил вас указать своим товарищам явно о тоне, с которым он передает сообщение *s*.

Для этого, если *s* содержит хотя бы один восклицательный знак, все буквы этого сообщения необходимо сделать заглавными. Иначе, все буквы следует сделать строчными.

Выведите результат этой операции.

**Входные данные**

В единственной строке вводится фраза *s*, состоящая из латинских букв и знаков препинания. Длина этой фразы не превосходит *20*.

**Выходные данные**

Выведите результат преобразования сообщения.

**Пример**

| **Входные данные** |
|--------------------|
| Do it! Now!        |

| **Выходные данные** |
|---------------------|
| DO IT! NOW!         |

| **Входные данные** |
|--------------------|
| Sorry, what?       |

| **Выходные данные** |
|---------------------|
| sorry, what?        |


Все тесты пройдены, задача сдана:
```py
s = input()
if s.count('!') > 0:
    print(s.upper())
else:
    print(s.lower())
```
