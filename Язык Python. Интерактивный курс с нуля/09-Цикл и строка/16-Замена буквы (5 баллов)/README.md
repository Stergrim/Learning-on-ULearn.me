# Замена буквы

<p align="center">
<b>Замена буквы</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам дана строка *s*, число *k* и символ *t*.

Ваша задача — заменить символ, стоящий по индексу *k* на символ *t*, а затем вывести измененную строку.

**Входные данные**

В первой строке вводится строка *s*, состоящая из латинских букв *(1≤∣s∣≤50)*.

Во второй и третьей строке вводятся число *k* и символ *t*, соответственно. *t* — латинская буква, а *k* удовлетворяет ограничениям: *1≤k≤∣s∣*.

**Выходные данные**

Выведите строку после замены буквы.

**Пример**

| **Входные данные** |
|--------------------|
| tall               |
| 2                  |
| e                  |

| **Выходные данные** |
|---------------------|
| tell                |


Все тесты пройдены, задача сдана:
```py
s = input()
k = int(input())
t = input()
print(s[:k-1] + t + s[k:])
```
