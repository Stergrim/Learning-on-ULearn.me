# Интересный запрос

<p align="center">
<b>Интересный запрос</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам дана строка ss и два интересных числа: *l* и *d*.

Вас просят вывести такую подстроку *s*, что она начинается с *l*-го символа строки, и при этом ее длина в точности равна *d*.

Гарантируется, что такая подстрока существует.

**Входные данные**

В первой строке вводится строка *s*, состоящая из латинских букв *(1≤∣s∣≤50)*.

Во второй и третьей строке вводятся числа *l* и *d*, соответственно *(1≤l,d≤∣s∣, l+d≤∣s∣+1)*. 

**Выходные данные**

Выведите требуемую подстроку.

**Пример**

| **Входные данные** |
|--------------------|
| doggo              |
| 1                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| dog                 |


Все тесты пройдены, задача сдана:
```py
s = input()
l = int(input())
d = int(input())
print(s[l - 1:l + d - 1])
```
