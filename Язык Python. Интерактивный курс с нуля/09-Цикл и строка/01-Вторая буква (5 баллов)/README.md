# Вторая буква

<p align="center">
<b>Вторая буква</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам вводится слово *s*, состоящее из трех букв. Выведите вторую букву этого слова.

**Входные данные**

В единственной строке вводится слово *s* из трех латинских букв.

**Выходные данные**

Выведите только один символ — вторую букву этого слова.

**Пример**

| **Входные данные** |
|--------------------|
| lol                |

| **Выходные данные** |
|---------------------|
| o                   |


Все тесты пройдены, задача сдана:
```py
s = input()
print(s[1])
```
