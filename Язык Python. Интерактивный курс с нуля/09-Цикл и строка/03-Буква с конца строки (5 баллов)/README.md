# Буква с конца строки

<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам вводится слово *s*. Выведите *k*-ю букву этого слова с **конца** строки.

**Входные данные**

В первой строке вводится слово *s*, состоящее из латинских букв. Длина *s* не превосходит *10*.

Во второй строке вводится целое число *k* *(1≤k≤∣s∣)*.

**Выходные данные**

Выведите только один символ — *k*-ю букву этого слова с конца строки.

**Пример**

| **Входные данные** |
|--------------------|
| flown              |
| 2                  |

| **Выходные данные** |
|---------------------|
| w                   |


Все тесты пройдены, задача сдана:
```py
s = input()
k = int(input())
print(s[-k])
```
