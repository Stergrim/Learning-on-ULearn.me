# Количество вопросов

<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вася сказал: *s*. Посчитайте количество вопросительных знаков, которые встречаются в *s*.

**Входные данные**

В единственной строке вводится фраза *s*, состоящая из латинских букв и знаков препинания. Длина этой фразы не превосходит *20*.

**Выходные данные**

Выведите единственное целое число — ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| What? Where? Why?  |

| **Выходные данные** |
|---------------------|
| 3                   |


Все тесты пройдены, задача сдана:
```py
s = input()
print(s.count('?'))
```
