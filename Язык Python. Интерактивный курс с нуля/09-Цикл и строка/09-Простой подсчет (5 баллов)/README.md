# Простой подсчет

<p align="center">
<b>Простой подсчет</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Потренируйтесь сделать обычный подсчет: скажите, сколько символов *t* содержится в строке *s*.

**Входные данные**

В первой строке вводится слово *s* из заглавных латинских букв. Длина *s* не превосходит *50*.

Во второй строке содержится заглавная латинская буква *t*.

**Выходные данные**

Выведите количество вхождений *t* в строку *s*.

**Пример**

| **Входные данные** |
|--------------------|
| BAKAMITAI          |
| A                  |

| **Выходные данные** |
|---------------------|
| 3                   |


Все тесты пройдены, задача сдана:
```py
s = input()
t = input()
print(s.count(t))
```
