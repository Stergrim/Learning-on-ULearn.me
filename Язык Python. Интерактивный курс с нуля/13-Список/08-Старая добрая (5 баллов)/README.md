# Старая добрая

<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Найдите сумму двух чисел.

**Входные данные**

В единственной строке вводятся два числа: *a,b* *(1≤a,b≤100)*.

**Выходные данные**

Выведите единственное целое число — сумму двух чисел.

**Пример**

| **Входные данные** |
|--------------------|
| 2 3                |

| **Выходные данные** |
|---------------------|
| 5                   |


Все тесты пройдены, задача сдана:
```py
a, b = map(int, input().split())
print(a + b)
```
