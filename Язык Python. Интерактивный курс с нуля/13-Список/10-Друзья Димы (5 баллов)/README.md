У Димы есть три друга, и он позвал их на вечеринку. Первый принес ему *a* конфет, второй — *b*, а третий — *c*.

У Димы возникла идея: конфеты от одного друга положить на стол, а конфеты остальных двух друзей спрятать у себя в комнате. Скажите, какое наибольшее число конфет Дима сможет спрятать?

**Входные данные**

В единственной строке вводятся три числа: *a,b,c* *(1≤a,b,c≤100)*.

**Выходные данные**

Выведите единственное целое число — максимальное количество конфет, которое Дима сможет спрятать.

**Пример**

| **Входные данные** |
|--------------------|
| 1 2 3              |

| **Выходные данные** |
|---------------------|
| 5                   |


Все тесты пройдены, задача сдана:
```py
a, b, c = map(int, input().split())
print(sum([a, b, c]) - min([a, b, c]))
```
