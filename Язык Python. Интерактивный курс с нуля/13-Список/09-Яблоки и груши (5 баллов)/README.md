В магазине продаются пакеты яблок и груш. В одном пакете *a* яблок или *b* груш.

Настю попросили купить домой не более *A* яблок и не более *B* груш, при этом взять их по максимуму. Сообщите, сколько пакетов Насте нужно взять домой.

**Входные данные**

В единственной строке вводятся четыре числа: *a,b,A,B* *(1≤a,b,A,B≤100)*.

**Выходные данные**

Выведите единственное целое число — количество пакетов.

**Пример**

| **Входные данные** |
|--------------------|
| 5 7 10 15          |

| **Выходные данные** |
|---------------------|
| 4                   |


Все тесты пройдены, задача сдана:
```py
a, b, A, B = map(int, input().split())
print(A // a + B // b)
```
