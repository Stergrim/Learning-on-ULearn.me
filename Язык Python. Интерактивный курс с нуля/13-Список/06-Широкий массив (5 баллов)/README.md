Шириной массива мы будем называть разность максимального и минимального элемента массива.

Дан массив *a1,a2,…,an*. Найдите его ширину.

**Входные данные**

В единственной строке вводятся числа *a1,a2,…,an* *(1≤n≤10, 1≤a<sub>i</sub>≤100)*.

**Выходные данные**

Выведите единственное целое число — ширину введенного массива.

**Пример**

| **Входные данные** |
|--------------------|
| 4 2 3              |

| **Выходные данные** |
|---------------------|
| 2                   |


Все тесты пройдены, задача сдана:
```py
a = list(map(int, input().split()))
print(abs(max(a) - min(a)))
```
