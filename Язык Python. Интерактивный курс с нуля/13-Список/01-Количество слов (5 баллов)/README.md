# Количество слов

<p align="center">
<b>Количество слов</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам будет дано предложение, записаное в одной строке. Ваша задача — вывести количество слов в нем, если известно, что слова разделены ровно одним пробелом.

**Входные данные**

В единственной строке вводится предложение *s*, состоящее исключительно из латинских букв и пробелов. Длина *s* не превосходит *100*.

**Выходные данные**

Выведите единственное целое число — количество слов в *s*.

**Пример**

| **Входные данные** |
|--------------------|
| First second third |

| **Выходные данные** |
|---------------------|
| 3                   |


Все тесты пройдены, задача сдана:
```py
s = input().split()
print(len(s))
```
