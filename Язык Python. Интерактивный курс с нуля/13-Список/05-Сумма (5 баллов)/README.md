# Сумма

<p align="center">

ограничение по времени на тест:

ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Дан массив *a1,a2,…,an*. Посчитайте сумму элементов данного массива.

**Входные данные**

В единственной строке вводятся числа *a1,a2,…,an* *(1≤n≤10, 1≤a<sub>i</sub>≤100)*.

**Выходные данные**

Выведите единственное целое число — сумму всех чисел.

**Пример**

| **Входные данные** |
|--------------------|
| 1 2 3              |

| **Выходные данные** |
|---------------------|
| 6                   |


Все тесты пройдены, задача сдана:
```py
a = list(map(int, input().split()))
print(sum(a))
```
