<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>

стандартный вывод <br>
</p>

Даны строчные буквы *a* и *b*. Скажите, насколько далеко они находятся друг от друга в unicode.

**Входные данные**

В двух строках вводятся символы *a* и *b*. Коды символов находятся в диапазоне от *32* до *125*.

**Выходные данные**

Выведите расстояние между символами *a* и *b* — разность их кодов.

**Пример**

| **Входные данные** |
|--------------------|
| A                  |
| a                  |

| **Выходные данные** |
|---------------------|
| 32                  |

| **Входные данные** |
|--------------------|
| a                  |
| A                  |

| **Выходные данные** |
|---------------------|
| 32                  |

Все тесты пройдены, задача сдана:
```py
a = input()
b = input()
print(abs(ord(a) - ord(b)))
```
