# Скачок

<p align="center">
<b>Скачок</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам дан символ *s* и число *x*. Выведите символ, который в unicode стоит на *x* позиций дальше, чем символ *s*.

**Входные данные**

В первой строке вводится символ *s*. Гарантируется, что код этого символа не меньше *32* и не больше *125*.

Во второй строке вводится число *x* — оно может быть как положительным, так и отрицательным. Гарантируется, что итоговый символ будет иметь код не меньше *32* и не больше *125*.

**Выходные данные**

Выведите единственный символ — ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| A                  |
| 32                 |

| **Выходные данные** |
|---------------------|
| a                   |

| **Входные данные** |
|--------------------|
| a                  |
| -32                |

| **Выходные данные** |
|---------------------|
| A                   |

Все тесты пройдены, задача сдана:
```py
s = input()
x = int(input())
print(chr((ord(s) + x)))
```
