# Порядок в словаре

<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>

стандартный вывод <br>
</p>

Вам даны две строки *a* и *b*. Верно ли, что строка *a* идет в словаре раньше, чем строка *b*? Обратите внимание, что в словаре не учитывается регистр!

**Входные данные**

В первой строке вводится *a* — слово из латинских букв.

Во второй строке вводится *b* — слово из латинских букв.

Гарантируется, что *1≤∣a∣,∣b∣≤10*. Гарантируется, что без учета регистра это равные строки.

**Выходные данные**

Выведите «YES», если *a* идет в словаре раньше, чем *b*, и «NO», иначе.

**Пример**

| **Входные данные** |
|--------------------|
| alien              |
| BUDDY              |

| **Выходные данные** |
|---------------------|
| YES                 |


Все тесты пройдены, задача сдана:
```py
a = input()
b = input()
if a.lower() < b.lower():
    print("YES")
else:
    print("NO")
```
