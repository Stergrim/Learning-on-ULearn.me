# Пересечение двух множеств

<p align="center">
<b>Пересечение двух множеств</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны два массива чисел *a1,a2,…,an* и *b1,b2,…,bn*. Найдите пересечение данных массивов и выведите количество элементов в этом пересечении.

**Входные данные**

В первой строке друг за другом вводятся числа *a1,a2,…,an* *(1≤n≤10, 1≤a<sub>i</sub>≤10)*, а во второй строке в таком же формате числа *b1,b2,…,Bn* *(1≤B<sub>i</sub>≤10)*

**Выходные данные**

Выведите единственное целое число — количество чисел в пересечении массивов чисел из входных данных.

**Пример**

| **Входные данные** |
|--------------------|
| 1 2 3 3            |
| 2 3 4              |

| **Выходные данные** |
|---------------------|
| 2                   |

| **Входные данные** |
|--------------------|
| 1 2 3              |
| 4 5 6              |

| **Выходные данные** |
|---------------------|
| 0                   |


Все тесты пройдены, задача сдана:
```py
a = set(map(int, input().split()))
b = set(map(int, input().split()))
c = a & b
print(len(c))
```
