ограничение по памяти на тест:

ввод:

стандартный вывод <br>

Вам дан массив *a1,a2,…,an*. Выведите количество различных чисел в этом массиве.

**Входные данные**

В единственной строке друг за другом вводятся числа *a1,a2,…,an* *(1≤n≤10, 1≤a<sub>i</sub>≤100)*.

**Выходные данные**

Выведите единственное целое число — количество различных чисел в массиве.

**Пример**

| **Входные данные** |
|--------------------|
| 1 2 3 2 1          |

| **Выходные данные** |
|---------------------|
| 3                   |

| **Входные данные** |
|--------------------|
| 2 2 3              |

| **Выходные данные** |
|---------------------|
| 2                   |


Все тесты пройдены, задача сдана:
```py
a = set(map(int, input().split()))
print(len(a))
```
