(1 –∏–∑ 1 –±–∞–ª–ª–∞)
    * üü¢ `'**' ''` (–ü—Ä–∞–≤–∏–ª—å–Ω—ã–π –æ—Ç–≤–µ—Ç: '*' '**')


```py

for i in range(3, 7):
    if i % 2 == 0:
        print('+', i, sep='', end='')
        ans += i
    else:
        print('-', i, sep='', end='')
        ans -= i
print('=', ans, sep='')
```
