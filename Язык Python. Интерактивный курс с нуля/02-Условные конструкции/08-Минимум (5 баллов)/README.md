стандартный вывод <br>

Даны два целых числа *X* и *Y*. Выведите наименьшее из них.

**Входные данные**

В первой строке дано целое положительное число *X*, во второй – число *Y* *(1≤X,Y≤100)*.

**Выходные данные**

Выведите наименьшее из двух чисел.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |
| 10                 |

| **Выходные данные** |
|---------------------|
| 3                   |

| **Входные данные** |
|--------------------|
| 31                 |
| 39                 |

| **Выходные данные** |
|---------------------|
| 31                  |


Все тесты пройдены, задача сдана:
```py
x = int(input())
y = int(input())

if x <= y:
    print(x)

if x > y:
    print(y)
```
