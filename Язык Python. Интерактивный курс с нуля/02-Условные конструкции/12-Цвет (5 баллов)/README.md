# Цвет

<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>

стандартный вывод <br>
</p>

Дана шахматная доска размером *N×N*. Требуется определить цвет клетки по ее координатам.

**Входные данные**

Даны три целых числа, каждое в отдельной строке — *N*, *x*, *y*. *(1≤N≤100)(1≤x,y≤N)*.

**Выходные данные**

Выведите *black*, если клетка с координатами *(x,y)* черная, или *white*, если белая. Считается, что клетка с координатами *(1,1)* — чёрная.

**Пример**

| **Входные данные** |
|--------------------|
| 8                  |
| 8                  |
| 8                  |

| **Выходные данные** |
|---------------------|
| black               |


Все тесты пройдены, задача сдана:
```py
n = int(input())
x = int(input())
y = int(input())
if (x - y) % 2 == 0:
    print("black")
else:
    print("white")
```
