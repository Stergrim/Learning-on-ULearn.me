Наверное, каждый сталкивался с «ошибкой 404» — когда запрашиваемая страница не найдена. Оказывается, есть огромное количество кодов, которые возвращает сервер (например, сайт), поясняя свой ответ за отправленный запрос. Коды всегда трехзначные, и мы будем рассматривать только коды от *200* до *599*.

Заучивать все коды не имеет смысла, поэтому вам предлагается написать следующую программу-подсказку по коду *x*:
- Если *x=200*, то подсказка: «OK» (Все прошло успешно);
- Если *x=404*, то подсказка: «Not Found» (Не найдено);
- Если *x=501*, то подсказка: «Not implemented» (Это еще не реализовано);
- Если *x* начинается с *2*, но не *200*, то подсказка: «Success» (В целом, все прошло хорошо);
- Если *x* начинается с *5*, но не *501*, то подсказка: «Server error» (Неполадки на стороне сервера, например, сайта);
- Остальное пока реализовывать не будем, поэтому выведем уже знакомый нам код: «501». 

Реализуйте такую программу!

**Входные данные**

В единственной строке вводится целое число *x* *(200≤x≤599)* — код состояния.

**Выходные данные**

Выведите ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 202                |

| **Выходные данные** |
|---------------------|
| Success             |

| **Входные данные** |
|--------------------|
| 404                |

| **Выходные данные** |
|---------------------|
| Not Found           |

| **Входные данные** |
|--------------------|
| 300                |

| **Выходные данные** |
|---------------------|
| 501                 |


Все тесты пройдены, задача сдана:
```py
x = int(input())
if x == 200:
    print("OK")
elif x == 404:
    print("Not Found")
elif x == 501:
    print("Not implemented")
elif x // 100 == 2:
    print("Success")
elif x // 100 == 5:
    print("Server error")
else:
    print("501")
```
