# Угадал?

<p align="center">
<b>Угадал?</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Витя загадал число *x*.

Саша — волшебник. Он пытается угадать это число. Для этого он прочитал какое-то заклинание, помахал руками, и в конце концов предположил, что Витя загадал число *y*.

Угадал ли Саша?

**Входные данные**

В первой строке вводится число *x* *(1≤x≤10)* — число, которое загадал Витя.

Во второй строке вводится число *y* *(1≤y≤10)* — число, которое назвал Саша..

**Выходные данные**

Выведите «CORRECT», если Саша угадал число. Иначе, ничего не выводите. Слово должно выводиться заглавными буквами, без кавычек.

**Пример**

| **Входные данные** |
|--------------------|
| 7                  |
| 7                  |

| **Выходные данные** |
|---------------------|
| CORRECT             |

| **Входные данные** |
|--------------------|
| 1                  |
| 4                  |

| **Выходные данные** |
|---------------------|
|                     |


Все тесты пройдены, задача сдана:
```py
x = int(input())
y = int(input())
if x == y:
    print("CORRECT")
```
