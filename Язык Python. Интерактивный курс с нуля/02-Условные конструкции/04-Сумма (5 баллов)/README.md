# Сумма

<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вася очень не любит нечетные числа. Да так, что он решил все нечетные числа, которые он встретит, превратить в четные, умножив их на *2*.

Оказалось, что в мире слишком много нечетных чисел, поэтому Вася решил написать программу, которая поможет ему в этом нелегком деле.

**Входные данные**

Дано целое число *X* *(1≤X≤100)*.

**Выходные данные**

Выведите число *Y* по следующему правилу: если *X* четно, то выводится то же число (оно Васе нравится). Если же число нечетное, то программа умножает это число на *2* и выводит результат.

**Пример**

| **Входные данные** |
|--------------------|
| 5                  |

| **Выходные данные** |
|---------------------|
| 10                  |

| **Входные данные** |
|--------------------|
| 2                  |

| **Выходные данные** |
|---------------------|
| 2                   |


Все тесты пройдены, задача сдана:
```py
x = int(input())
if x % 2 == 1:
    x *= 2

print(x)
```
