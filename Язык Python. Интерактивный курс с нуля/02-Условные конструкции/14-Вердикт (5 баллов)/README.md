# Вердикт

<p align="center">
<b>Вердикт</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Саша пишет свою тестирующую систему для задач по программированию, и ему нужно, чтобы в ней осуществлялась проверка ограничений по времени и по памяти.

Для первой версии своей системы он решил установить для всех задач ограничение по времени в *2000* мс, а по памяти — в *64* МБ. Это значит, что программа может работать до *2000* мс (включительно) и потреблять память до *64* МБ (включительно).

Сеня попросил вас написать программу, которая будет по времени работы *T* и потребляемой памяти *M* делать следующее:
- Если программа не укладывается в ограничения по памяти, вывести «ML».
- Иначе, если программа не укладывается в ограничения по времени, вывести «TL».
- Иначе вывести «OK». 

Выводимую строку будем называть вердиктом.

Ваша задача — вывести правильный вердикт по заданным *T* и *M*.

**Входные данные**

В первой строке вводится время работы программы *T* в мс *(1≤T≤5000)*.

Во второй строке вводится потребляемая память *M* в МБ *(1≤M≤300)*.

**Выходные данные**

Выведите вердикт на основании введенных *T* и *M*.

**Пример**

| **Входные данные** |
|--------------------|
| 150                |
| 3                  |

| **Выходные данные** |
|---------------------|
| OK                  |

| **Входные данные** |
|--------------------|
| 5000               |
| 300                |

| **Выходные данные** |
|---------------------|
| ML                  |


Все тесты пройдены, задача сдана:
```py
t = int(input())
m = int(input())
if m > 64:
    print("ML")
elif t > 2000:
    print("TL")
else:
    print("OK")
```
