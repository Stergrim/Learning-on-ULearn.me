Дано число *X*. Если оно делится на *2*, то выведите результат деления *X* на *2*. Иначе выведите это же число.

**Входные данные**

Дано целое положительное число *X* *(1≤x≤100)*.

**Выходные данные**

Выведите ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 5                  |

| **Выходные данные** |
|---------------------|
| 5                   |

| **Входные данные** |
|--------------------|
| 6                  |

| **Выходные данные** |
|---------------------|
| 3                   |


Все тесты пройдены, задача сдана:
```py
x = int(input())
if x % 2 == 0:
    x //= 2

print(x)
```
