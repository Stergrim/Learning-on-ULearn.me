# Обрезка

<p align="center">
<b>Обрезка</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Дано число *A*. Если оно больше *50*, выведите *50*, иначе выведите исходное число.

**Входные данные**

Дано целое положительное число *A* *(1≤A≤100)*.

**Выходные данные**

Выведите число *A*, если *A≤50*, иначе выведите *50*.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |

| **Выходные данные** |
|---------------------|
| 3                   |

| **Входные данные** |
|--------------------|
| 92                 |

| **Выходные данные** |
|---------------------|
| 50                  |


Все тесты пройдены, задача сдана:
```py
a = int(input())
if a > 50:
    a = 50
	
print(a)
```
