Вовочка написал программу, которая принимает на вход *2* числа *a* и *b*, а на выход дает число *c*, которое вычисляется как *c=a+b*.

Вовочка начал тестировать программу, которая, к его удивлению, давала правильный ответ лишь изредка. Вдруг ему надоело делать это все руками, и он попросил вас написать тестирующую программу.

**Входные данные**

На вход даны три целых числа — *a*, *b* и *c* *(0≤a,b,c≤100)*, каждое из которых находится в отдельной строке.

**Выходные данные**

Если *c=a+b*, выведите *YES*, иначе выведите *NO*.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |
| 10                 |
| 13                 |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| 1                  |
| 2                  |
| 2                  |

| **Выходные данные** |
|---------------------|
| NO                  |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
c = int(input())
if c == a + b:
    print("YES")
else:
    print("NO")
```
