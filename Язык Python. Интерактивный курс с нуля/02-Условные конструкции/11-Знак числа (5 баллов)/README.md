# Знак числа

<p align="center">
<b>Знак числа</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам дано число *n*. Вам нужно вывести его знак: *−*, если число отрицательное; *+*, если положительное; *0*, если число равно нулю.

**Входные данные**

В единственной строке дано целое число *n* *(−100≤n≤100)*.

**Выходные данные**

Выведите ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |

| **Выходные данные** |
|---------------------|
| +                   |

| **Входные данные** |
|--------------------|
| -5                 |

| **Выходные данные** |
|---------------------|
| -                   |

| **Входные данные** |
|--------------------|
| 0                  |

| **Выходные данные** |
|---------------------|
| 0                   |


Все тесты пройдены, задача сдана:
```py
a = int(input())
if a > 0:
    print('+')
else:
    if a < 0:
        print('-')
    else:
        print('0')
```
