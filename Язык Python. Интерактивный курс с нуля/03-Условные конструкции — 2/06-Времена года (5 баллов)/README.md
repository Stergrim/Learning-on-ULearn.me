# Времена года

<p align="center">
<b>Времена года</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Дан номер месяца *X* *(1≤X≤12)*. В зависимости от этого номера выведите время года, в котором находится этот месяц.

**Входные данные**

Вводится целое положительное число *X*.

**Выходные данные**

Выведите время года, в котором находится месяц под данным номером, на английском языке. Переводы времен года на английский смотрите в примечании ниже.

**Пример**

| **Входные данные** |
|--------------------|
| 6                  |

| **Выходные данные** |
|---------------------|
| summer              |

**Примечание**

Доступные варианты: *summer* (лето), *autumn* (осень), *winter* (зима), *spring* (весна).

Все тесты пройдены, задача сдана:
```py
x = int(input())
if x > 2 and x < 6:
    print("spring")
elif x > 5 and x < 9:
    print("summer")
elif x > 8 and x < 12:
    print("autumn")
else:
    print("winter")
```
