# Агент X

<p align="center">
<b>Агент X</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Подпись агента *X* — двузначное число, у которого ровно одна цифра равна пятерке.

Скажите, могло ли двузначное число aa быть подписью агента *X*?

Напомним, что у двузначных чисел первая цифра равна *a // 10*, а вторая равна *a % 10*.

**Входные данные**

В единственной строке вводится целое число *a* *(10≤a≤99)*.

**Выходные данные**

Выведите «YES», если *a* может оказаться подписью агента *X*, и «NO», иначе.

**Пример**

| **Входные данные** |
|--------------------|
| 75                 |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| 42                 |

| **Выходные данные** |
|---------------------|
| NO                  |


Все тесты пройдены, задача сдана:
```py
x = int(input())
if x // 10 == 5:
    if  x % 10 != 5:
        print("YES")
    else:
        print("NO")
elif x % 10 == 5:
    if  x // 10 != 5:
        print("YES")
    else:
        print("NO")
else:
    print("NO")
```
