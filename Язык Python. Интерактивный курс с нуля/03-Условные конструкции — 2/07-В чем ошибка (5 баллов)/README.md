Вова написал программу, которая принимает два целых числа *A* и *B* и возвращает число *C*, которое рассчитывается по формуле *C=A+B*.

Во время тестирования Вова заметил, что программа работает неправильно. У него появилось предположение, что в некоторых случаях программа выводит *a−b* или *b−a* вместо *a+b*. Он просит вас написать программу, которая поможет ему протестировать свою.

**Входные данные**

На вход подаются три целых числа в разных строках: *a*, *b* и *c*. *(1≤a,b,c≤100)*

**Выходные данные**

Выведите *+*, если *c=a+b*. Выведите *−*, если *c=a−b* или *c=b−a*. Иначе выведите *:(*.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |
| 10                 |
| 13                 |

| **Выходные данные** |
|---------------------|
| +                   |

| **Входные данные** |
|--------------------|
| 2                  |
| 1                  |
| 1                  |

| **Выходные данные** |
|---------------------|
| -                   |

| **Входные данные** |
|--------------------|
| 4                  
| 10                 |
| 9                  |

| **Выходные данные** |
|---------------------|
| :(                  |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
c = int(input())
if c == a + b:
    print('+')
elif c == a - b or c == b - a:
    print('-')
else:
    print(":(")
```
