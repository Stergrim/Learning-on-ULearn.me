# Транспортировка

<p align="center">
<b>Транспортировка</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вове дали *2* числа *a* и *b*. Ему сказали, что *a* делится на *b*, но, пока они шли до Вовы, число *b* могло поцарапаться, то есть стать больше на единицу или меньше на единицу (но могло и не измениться).

Вова хочет проверить, не обманули ли его отправители, и просит вас написать программу, которая помогла бы ему это сделать.

**Входные данные**

Даны два натуральных числа *a* и *b* *(1≤a,b≤100)*.

**Выходные данные**

Выведите *YES*, если *a* действительно могло делиться на *b* до перевозки, иначе выведите *NO*.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |
| 1                  |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| 4                  |
| 10                 |

| **Выходные данные** |
|---------------------|
| NO                  |

| **Входные данные** |
|--------------------|
| 2                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| YES                 |


**Примечание**

В последнем примере число *1* поцарапалось и превратилось в *2*.

Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
if a % b == 0 or a % (b - 1) == 0 or a % (b + 1) == 0:
    print("YES")
else:
    print("NO")
```
