стандартный ввод <br>

стандартный вывод <br>

Коля решил покататься вместе со своими друзьями на автобусе и оплатить им проезд. Известно, что один билет стоит *X* рублей *(0<X≤20)*, а поедут вместе с Колей *N* друзей *(0<N≤10)*. Сколько суммарно придётся заплатить денег за такую поездку? Разумеется, за себя Коля тоже должен заплатить.

**Входные данные**

В первой строке дано число *X* – стоимость проезда в автобусе, во второй – число *N*, количество друзей Коли.

**Выходные данные**

Выведите итоговую стоимость поездки Коли с друзьями и количество билетов, которое потребуется приобрести.

**Пример**

| **Входные данные** |
|--------------------|
| 10                 |
| 5                  |

| **Выходные данные** |
|---------------------|
| 60 6                |

**Примечание**

Обратите внимание, что оба числа надо вывести в одну строку.

Все тесты пройдены, задача сдана:
```py
x = int(input())
n = int(input())
print((n + 1)*x, n + 1)
```
