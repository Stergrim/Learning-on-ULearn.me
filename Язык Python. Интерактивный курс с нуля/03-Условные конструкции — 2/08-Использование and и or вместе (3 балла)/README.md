# Использование and и or вместе

```py
a = ___
if a > 100 or a <= 40 and a % 2 == 0:
    print('YES')
else:
    print('NO')
```

1. Дана программа. Чему может равнятся переменная аа, чтобы программа вывела YES? (1 из 1 балла)
   * ✅ **24** (Правильно!)
   * ❌ **29**
   * ❌ **56**
   * ❌ **1**
   * ✅ **137** (Правильно!)


```py
a = 34
b = 77
if b < 40 ___ a % 2 != 0 ___ b <= 100:
    print('YES')
```

2. Дана программа с пропусками. Выберите, какие логические операторы и куда нужно расставить, чтобы программа вывела YES. (1 из 1 балла)

   * ✅ **сначала - or, потом or** (Правильно!)
   * ❌ **сначала - and, потом and**
   * ❌ **сначала - or, потом and**
