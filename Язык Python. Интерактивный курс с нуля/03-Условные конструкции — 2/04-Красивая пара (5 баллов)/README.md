# Красивая пара

<p align="center">
<b>Красивая пара</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

У Васи есть два числа: *a* (первое число) и *b* (второе число). Первое число красивое, если оно делится на семь.

Красота же второго числа зависит от первого. Если первое число красивое, то второе красивое, если делится на три. Если же первое число не красивое, то и второе красивым быть не может.

Вася сам запутался в своем понимании красоты. Напишите ему программу, которая скажет, сколько у него красивых чисел.

**Входные данные**

В первой строке вводится целое число *a* *(1≤a≤100)* — первое число в паре.

Во второй строке вводится целое число *b* *(1≤b≤100)* — второе число в паре.

**Выходные данные**

Выведите единственное целое число — количество красивых чисел у Васи.

**Пример**

| **Входные данные** |
|--------------------|
| 7                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| 2                   |

| **Входные данные** |
|--------------------|
| 8                  |
| 4                  |

| **Выходные данные** |
|---------------------|
| 0                   |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
count = 0
if a % 7 == 0:
    count += 1
    if b % 3 == 0:
        count += 1

print(count)
```
