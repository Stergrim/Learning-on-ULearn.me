# Городская больница

<p align="center">
<b>Городская больница</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

В маленьком городе Питонграде *3* района и всего одна больница, которая не работает по воскресеньям. У больницы есть расписание работы для каждого района. Жители районов могут попасть в больницу по следующим правилам: жители первого района могут приходить только по чётным дням, жители второго – только по нечётным, а жители третьего только по тем числам, которые делятся на *3*.

Вам нужно написать программу, которая по заданному номеру района и числу поможет жителю города понять, сможет ли он попасть на прием в указанный день или нет. Напоминаем, что каждый седьмой день (в воскресенье) больница не работает.

**Входные данные**

В первой строке вводится одно из чисел *1*, *2* или *3* – номер района. Во второй строке вводится целое число *d* *(1≤d≤31)* – дата приема.

**Выходные данные**

Выведите *"yes"* (без кавычек), если в указанный день житель города сможет попасть на прием, *"no"* (без кавычек) – если не сможет.

**Пример**

| **Входные данные** |
|--------------------|
| 1                  |
| 6                  |

| **Выходные данные** |
|---------------------|
| yes                 |

| **Входные данные** |
|--------------------|
| 2                  |
| 7                  |

| **Выходные данные** |
|---------------------|
| no                  |

| **Входные данные** |
|--------------------|
| 3                  |
| 15                 |

| **Выходные данные** |
|---------------------|
| yes                 |


Все тесты пройдены, задача сдана:
```py
n = int(input())
d = int(input())
if ((n == 1 and d % 2 == 0) or (n == 2 and d % 2 != 0) or (n == 3 and d % 3 == 0)) and d % 7 != 0:
    print("yes")
else:
    print("no")
```
