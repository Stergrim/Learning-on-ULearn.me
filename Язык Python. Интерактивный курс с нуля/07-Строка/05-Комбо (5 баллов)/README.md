# Комбо

<p align="center">
<b>Комбо</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Чтобы получить комбо в одной известной игре, Саше надо нажать *n* раз стрелочку "вправо", а потом стрелочку "вверх".

Саша хочет сделать *m* комбо подряд. Помогите ему: напишите последовательность действий для этого, используя букву «r» для обозначения команды «нажать вправо» и «u» для нажатия вверх.

**Входные данные**

В первой строке вводится целое число *n* *(1≤n≤5)*.

Во второй строке вводится целое число *m* *(1≤m≤5)*.

**Выходные данные**

Выведите единственную строку без пробелов — *m* комбо.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |
| 2                  |

| **Выходные данные** |
|---------------------|
| rrrurrru            |


Все тесты пройдены, задача сдана:
```py
n = int(input())
m = int(input())
print(('r' * n + 'u') * m)
```
