Преподаватель хочет поставить *n* ученикам пятерку и написать «Молодец!». Но пятерок так много, что он обратился за помощью к вам. Ему нужно nn раз через пробел написать слово «perfect» (без кавычек). Помогите ему!

**Входные данные**

В единственной строке вводится число *n* *(1≤n≤50)*.

**Выходные данные**

Выведите *n* слов «perfect».

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |

| **Выходные данные**     |
|-------------------------|
| perfect perfect perfect |


Все тесты пройдены, задача сдана:
```py
n = int(input())
print("perfect " * n)
```
