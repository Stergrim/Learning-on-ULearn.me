Вам даны имена *A* и *B*. Выведите выражение «A and B are AB together», подставив вместо *A* и *B* введенные имена. Да, вам нужно их сцепить друг за другом.

**Входные данные**

В двух строках вводятся имена: строки *A* и *B* длины до *10*. Строки состоят исключительно из букв латинского алфавита.

**Выходные данные**

Выведите требуемое выражение.

**Пример**

| **Входные данные** |
|--------------------|
| Joahn              |
| Jonny              |

| **Выходные данные**                     |
|-----------------------------------------|
| Joahn and Jonny are JoahnJonny together |


Все тесты пройдены, задача сдана:
```py
a = input()
b = input()
print(a + " and " + b + " are " + a + b + " together")
```
