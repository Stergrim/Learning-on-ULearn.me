# Министерство правды

<p align="center">
<b>Министерство правды</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

*Ложь, повторенная тысячу раз, становится правдой.*

Мы, конечно, против такой подмены, поэтому выведите слово «False» (без кавычек) **только десять** раз, при этом поставьте пробелы между словами. Ничего страшного, если после последнего слова будет стоять лишний пробел.

**Входные данные**

Вашей программе на вход ничего не подается.

**Выходные данные**

Выведите *10* раз слово «False».

Все тесты пройдены, задача сдана:
```py
print("False " * 10)
```
