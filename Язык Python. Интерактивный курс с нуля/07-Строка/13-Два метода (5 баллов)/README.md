# Два метода

<p align="center">
<b>Два метода</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны числа *A* и *B*. Пусть *C* — это результат сцепления (конкатенации) *A* и *B*, а *D* — результат их сложения. Выведите *C−D*.

**Входные данные**

Вашей программе в двух строках вводятся числа *A* и *B* *(1≤A,B≤100)*.

**Выходные данные**

Выведите единственное целое число — ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 5                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| 45                  |


Все тесты пройдены, задача сдана:
```py
a = input()
b = input()
c = int(a + b)
d = int(a) + int(b)
print(c - d)
```
