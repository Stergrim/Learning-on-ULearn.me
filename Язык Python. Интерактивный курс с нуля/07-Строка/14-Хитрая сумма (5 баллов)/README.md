# Хитрая сумма

<p align="center">
<b>Хитрая сумма</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Сложите два длинных числа: первое состоит из *n* цифр *a*, записанных друг за другом, а второе состоит из *m* цифр *b*, записанных друг за другом.

**Входные данные**

В четырех строках вводятся числа *n,a,m,b* *(1≤a,b≤9, 1≤n,m≤100)*.

**Выходные данные**

Выведите единственное целое число — сумму двух чисел.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |
| 1                  |
| 2                  |
| 2                  |

| **Выходные данные** |
|---------------------|
| 133                 |


Все тесты пройдены, задача сдана:
```py
n = int(input())
a = input()
m = int(input())
b = input()
print(int(n * a) + int(m * b))
```
