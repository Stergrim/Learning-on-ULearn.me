В гости к Маше зашел друг с именем *s*, но Маша не знает, как правильно поздороваться. Она обратилась к вам. Выведите ей простой совет: строку «Hello, », а за ней имя друга.

**Входные данные**

В единственной строке вводится имя *s*, состоящее только из латинских букв *(1≤∣s∣≤10)*.

**Выходные данные**

Выведите требуемое приветствие.

**Пример**

| **Входные данные** |
|--------------------|
| Vasya              |

| **Выходные данные** |
|---------------------|
| Hello, Vasya        |

**Примечание**

Выражение ∣*s*∣ обозначает длину строки *s*. То есть по условию гарантируется, что длина строки *s* находится в диапазоне от одного до десяти.

Все тесты пройдены, задача сдана:
```py
s = input()
print("Hello, " + s)
```
