Вам даны имена *A* и *B*. Выведите выражение «love: A B», подставив вместо *A* и *B* введенные имена.

**Входные данные**

В двух строках вводятся имена: строки *A* и *B* длины до *10*. Строки состоят исключительно из букв латинского алфавита.

**Выходные данные**

Выведите требуемое выражение.

**Пример**

| **Входные данные** |
|--------------------|
| Sasha              |
| Masha              |

| **Выходные данные** |
|---------------------|
| love: Sasha Masha   |


Все тесты пройдены, задача сдана:
```py
a = input()
b = input()
print("love: " + a + ' ' + b)
```
