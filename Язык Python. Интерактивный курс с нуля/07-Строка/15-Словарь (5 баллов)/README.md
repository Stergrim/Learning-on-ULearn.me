# Словарь

<p align="center">
<b>Словарь</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны две различные строки, состоящие исключительно из заглавных латинских букв: *s1* и *s2*. Правда ли, что *s1* идет в словаре раньше, чем *s2*.

**Входные данные**

В первой строке вводистся *s1*, а во второй — *s2* *(1≤∣s1∣,∣s2∣≤10, s1≠s2)*.

**Выходные данные**

Выведите «YES» (без кавычек), если слово *s1* идет в словаре раньше, чем *s2*, иначе, выведите «NO».

**Пример**

| **Входные данные** |
|--------------------|
| APPLE              |
| BANANA             |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| MIRRORING          |
| MIRROR             |

| **Выходные данные** |
|---------------------|
| NO                  |


Все тесты пройдены, задача сдана:
```py
s1 = input()
s2 = input()
if s1 < s2:
    print("YES")
else:
    print("NO")
```
