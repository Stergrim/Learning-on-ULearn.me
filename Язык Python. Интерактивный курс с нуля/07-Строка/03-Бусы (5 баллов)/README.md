Бусами длины nn назовем такую строку: *«-O-O-...-O-O-»*. В этой строке *n* заглавных латинских букв *«O»* и *n+1* минус.

Например, бусы длины *1* выглядят так: *«-O-»*. Бусы длины *3* выглядят так: *«-O-O-O-»*.

**Ваша задача:** вывести бусы длины *40*.

В отличие от пробелов, минусы строго учитываются проверяющей системой, поэтому выведите в точности то, что требуется.

**Входные данные**

Вашей программе на вход ничего не подается.

**Выходные данные**

Выведите только одну строку - бусы длины *40*.

Все тесты пройдены, задача сдана:
```py
print("-O" * 40 + '-')
```
