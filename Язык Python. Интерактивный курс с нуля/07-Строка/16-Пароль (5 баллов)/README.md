# Пароль

<p align="center">
<b>Пароль</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

На вашем сайте есть пользователь Коля, пароль от его аккаунта — строка *c*.

Коля забыл пароль. Сейчас у него есть две строки *a* и *b*, и он в качестве пароля хочет попробовать ввести *a*, *b*, *a+b* и *b+a*. Скажите, окажется ли правильным какой-то из этих вариантов?

**Входные данные**

В первой строке вводится строка *a*, во второй — *b*, а в третьей — *c*. Строки могут состоять из различных символов. Длина строк не превосходит *10*.

**Выходные данные**

Выведите «YES» (без кавычек), если Коля сможет угадать пароль, иначе выведите «NO».

**Пример**

| **Входные данные** |
|--------------------|
| kek                |
| lol                |
| lolkek             |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| first              |
| second             |
| third              |

| **Выходные данные** |
|---------------------|
| NO                  |


Все тесты пройдены, задача сдана:
```py
a = input()
b = input()
c = input()
if a == c or b == c or a + b == c or b + a == c:
    print("YES")
else:
    print("NO")
```
