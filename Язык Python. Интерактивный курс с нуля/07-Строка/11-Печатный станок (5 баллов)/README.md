# Печатный станок

<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

У вас есть заказ распечатать *n* копий некоторой строки *s*. Выведите результат выполнения заказа. Разделяйте копии пробелами.

**Входные данные**

В первой строке вводится число *n* *(1≤n≤10)*.

Во второй строке вводится текст *s* *(1≤∣s∣≤100)*.

**Выходные данные**

Выведите требуемое выражение.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |
| Hello              |

| **Выходные данные** |
|---------------------|
| Hello Hello Hello   |


Все тесты пройдены, задача сдана:
```py
n = int(input())
s = input()
print((s + ' ') * n)
```
