Песня одной известной группы «Буря» состоит из трёх куплетов. Каждый куплет состоит из тринадцати слов «thunder» и двух идущих за ними одинаковых строчек «thunder feel the thunder lightning and the thunder». Без всяких знаков препинания, маленьких букв, разделяя слова исключительно пробелами, выведите текст песни «Буря».

**Входные данные**

Вашей программе на вход ничего не подаётся.

**Выходные данные**

Выведите текст песни «Буря».

Все тесты пройдены, задача сдана:
```py
verse = "thunder " * 13 + "thunder feel the thunder lightning and the thunder " * 2
print(verse * 3)
```
