# Оценка

<p align="center">
<b>Оценка</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

В Берляндии корректная оценка в аттестате должна быть числом или строкой, состоящий только из маленьких букв и знаков препинания, пробелов или цифр, начинающейся с фразы: «result: » (обязательно должен стоять пробел после двоеточия).

У Васи в аттестате стоит оценка *s*. Проверьте, корректна ли она.

**Входные данные**

В единственной строке вводится текст *s*, состоящий из не более чем *10* символов.

**Выходные данные**

Выведите «YES», если *s* — корректная оценка, и «NO», иначе.

**Пример**

| **Входные данные** |
|--------------------|
| result: ok         |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| 80                 |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| ok                 |

| **Выходные данные** |
|---------------------|
| NO                  |

| **Входные данные** |
|--------------------|
| result: WA         |

| **Выходные данные** |
|---------------------|
| NO                  |

Все тесты пройдены, задача сдана:
```py
s = input()
if s.isdigit() or (s.startswith("result: ") and s.islower()):
    print("YES")
else:
    print("NO")
```
