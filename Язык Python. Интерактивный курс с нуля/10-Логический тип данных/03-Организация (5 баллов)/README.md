# Организация

<p align="center">
<b>Организация</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

В Берлянии запрещено в качестве имен организаций использовать строчные буквы. Другие символы использовать можно.

Проверьте, может ли открыться организация с названием *s*?

**Входные данные**

В единственной строке вводится текст *s*, состоящий из не более чем *10* символов. Гарантируется, что *s* состоит только из заглавных и строчных латиснких букв.

**Выходные данные**

Выведите «YES», если *s* — корректная организация, и «NO», иначе.

**Пример**

| **Входные данные** |
|--------------------|
| KEK                |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| organiss           |

| **Выходные данные** |
|---------------------|
| NO                  |


Все тесты пройдены, задача сдана:
```py
s = input()
if s.isupper():
    print("YES")
else:
    print("NO")
```
