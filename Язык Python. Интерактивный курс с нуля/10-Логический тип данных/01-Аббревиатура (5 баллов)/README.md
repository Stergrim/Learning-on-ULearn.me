# Аббревиатура

<p align="center">
<b>Аббревиатура</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Аббревиатурой назовем строку, состояющую исключительно из заглавных букв латинского алфавита.

Проверьте, является ли строка *s* аббревиатурой.

**Входные данные**

В единственной строке вводится текст *s*, состоящий из не более чем *10* символов.

**Выходные данные**

Выведите «YES», если *s* — аббревиатура, и «NO», иначе.

**Пример**

| **Входные данные** |
|--------------------|
| GGWP               |

| **Выходные данные** |
|---------------------|
| YES                 |

| **Входные данные** |
|--------------------|
| google             |

| **Выходные данные** |
|---------------------|
| NO                  |


Все тесты пройдены, задача сдана:
```py
s = input()
if s.isupper():
    print("YES")
else:
    print("NO")
```
