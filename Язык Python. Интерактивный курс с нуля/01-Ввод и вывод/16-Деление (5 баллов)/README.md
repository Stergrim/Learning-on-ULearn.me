# Деление

<p align="center">
<b>Деление</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны два целых числа *A* *(0≤A≤10)* и *B* *(1≤B≤10)*. Выведите сначала результат деления *A* на *B*, затем остаток от такого деления.

**Входные данные**

В первой строке дано целое число *A*, во второй – целое число *B*.

**Выходные данные**

Выведите сначала результат **целочисленного** деления *A* на *B* нацело. Во второй строке выведите остаток от деления *A* на *B*.

**Пример**

| **Входные данные** |
|--------------------|
| 10                 |
| 2                  |

| **Выходные данные** |
|---------------------|
| 5                   |
| 0                   |

| **Входные данные** |
|--------------------|
| 9                  |
| 2                  |

| **Выходные данные** |
|---------------------|
| 4                   |
| 1                   |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
print(a // b)
print(a % b)
```
