# Вова и книжки по математике

<p align="center">
<b>Вова и книжки по математике</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вова любит читать книжки по математике. У него всего *3* книжки, но, так как книжки слишком сложные, Вова не может читать одну книжку слишком долго, поэтому он читает их с перерывами. Первую книжку Вова читает по *30* минут за раз, вторую – по *45* минут, третью – по *60*. Вам известно, что Вова за эти сутки садился за первую книжку *A* раз, за вторую – *B* раз, за третью – *C* раз.

Вычислите, сколько времени за последние сутки Вовочка НЕ читал книжки по математике.

**Входные данные**

В первой строке дано целое число *A*, во второй строке – целое число *B*, в третьей – целое число *C*.

Гарантируется, что общее время чтения не превышает 24 часа.

**Выходные данные**

Выведите в первой строке количество часов, а во второй строке количество минут, которое Вовочка провел в последние сутки НЕ за чтением книг. Нужно вывести не общее количество минут, а количество минут, не входящих в полные часы.

**Пример**

| **Входные данные** |
|--------------------|
| 12                 |
| 4                  |
| 1                  |

| **Выходные данные** |
|---------------------|
| 14                  |
| 0                   |

| **Входные данные** |
|--------------------|
| 0                  |
| 0                  |
| 24                 |

| **Выходные данные** |
|---------------------|
| 0                   |
| 0                   |

**Примечание**

В сутках *24* часа, в каждом часе *60* минут.

Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
c = int(input())
d = a*30 + b*45 + c*60
print((24*60 - d) // 60)
print((24*60 - d) % 60)
```
