# Прочти число

<p align="center">
<b>Прочти число</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Дано число *A*. Ваша программа должна прочитать его и вывести обратно на экран.

**Входные данные**

В единственной строке дано число *A*, не превосходящее *10*.

**Выходные данные**

Выведите число *A*.

**Пример**

| **Входные данные** |
|--------------------|
| 6                  |

| **Выходные данные** |
|---------------------|
| 6                   |


Все тесты пройдены, задача сдана:
```py
a = int(input())
print(a)
```
