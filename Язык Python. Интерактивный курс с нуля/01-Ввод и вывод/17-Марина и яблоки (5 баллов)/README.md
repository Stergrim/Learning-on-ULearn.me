# Марина и яблоки

<p align="center">
<b>Марина и яблоки</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

У Марины есть *3* друга и *3* корзинки с яблоками. В первой корзинке лежат *A* яблок, во второй – *B*, в третьей – *C*. Марина решила поделить яблоки между друзьями поровну.

Делёж происходит так: Марина берёт первую корзину и раздаёт яблоки из неё поровну, после чего, чтобы никто не обижался, оставшиеся яблоки она съедает сама. Далее Марина переходит ко второй корзине и точно так же делит яблоки из неё. Затем аналогично делятся яблоки из третьей корзины. Оставшиеся из каждой корзины яблоки не складываются вместе и сразу после раздачи съедаются Мариной.

В первой строке выведите количество яблок, которое получит каждый из друзей, а во второй – количество яблок, которые съест Марина.

**Входные данные**

В первой строке дано целое число *A*, во второй строке – число *B*, в третьей – *C*.

Каждое число лежит в диапазоне от *1* до *100*.

**Выходные данные**

Выведите ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 5                  |
| 4                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| 3                   |
| 3                   |

| **Входные данные** |
|--------------------|
| 1                  |
| 2                  |
| 1                  |

| **Выходные данные** |
|---------------------|
| 0                   |
| 4                   |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
c = int(input())
print(a // 3 + b // 3 + c // 3)
print(a % 3 + b % 3 + c % 3)
```
