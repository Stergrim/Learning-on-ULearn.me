# Сумма трёх чисел

<p align="center">
<b>Сумма трёх чисел</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны три целых числа: *A*, *B* и *C*. Каждое число от *0* до *10*. Выведите их сумму.

**Входные данные**

В первой строке дано целое число *A*, во второй – целое число *B*, в третьей – целое число *C*.

**Выходные данные**

Выведите сумму чисел *A*, *B* и *C*.

**Пример**

| **Входные данные** |
|--------------------|
| 1                  |
| 2                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| 6                   |

| **Входные данные** |
|--------------------|
| 5                  |
| 6                  |
| 2                  |

| **Выходные данные** |
|---------------------|
| 13                  |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
c = int(input())
print(a + b + c)
```
