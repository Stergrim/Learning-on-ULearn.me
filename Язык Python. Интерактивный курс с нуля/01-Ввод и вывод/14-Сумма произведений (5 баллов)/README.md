# Сумма произведений

<p align="center">
<b>Сумма произведений</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны три числа: *A*, *B* и *C*. Каждое из чисел от *0* до *10*. Выведите сумму их попарных произведений, то есть сумму *3* слагаемых: *A⋅B*, *A⋅C*, *B⋅C*.

**Входные данные**

В первой строке дано целое число *A*, во второй – *B*, в третьей – *C*.

**Выходные данные**

Выведите сумму попарных произведений чисел.

**Пример**

| **Входные данные** |
|--------------------|
| 2                  |
| 3                  |
| 4                  |

| **Выходные данные** |
|---------------------|
| 26                  |

| **Входные данные** |
|--------------------|
| 1                  |
| 5                  |
| 6                  |

| **Выходные данные** |
|---------------------|
| 41                  |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
c = int(input())
print(a*b + a*c + b*c)
```
