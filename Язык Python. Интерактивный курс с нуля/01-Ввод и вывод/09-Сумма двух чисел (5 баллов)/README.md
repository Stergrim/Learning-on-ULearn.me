# Сумма двух чисел

<p align="center">
<b>Сумма двух чисел</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны два целых числа *A* и *B*, каждое от *0* до *10*. Выведите их сумму.

**Входные данные**

В первой строке дано целое число *A*, во второй строке – целое число *B*.

**Выходные данные**

Выведите сумму чисел *A* и *B*.

**Пример**

| **Входные данные** |
|--------------------|
| 1                  |
| 1                  |

| **Выходные данные** |
|---------------------|
| 2                   |

| **Входные данные** |
|--------------------|
| 2                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| 5                   |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
print(a + b)
```
