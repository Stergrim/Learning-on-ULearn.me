# Произведение суммы и числа

<p align="center">
<b>Произведение суммы и числа</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны три целых числа: *A*, *B* и *C*. Каждое число от *0* до *10*. Выведите *(A+B)∗C*.

**Входные данные**

В первой строке дано целое число *A*, во второй – целое число *B*, в третьей – целое число *C*.

**Выходные данные**

Выведите результат выражения *(A+B)∗C*.

**Пример**

| **Входные данные** |
|--------------------|
| 1                  |
| 2                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| 9                   |

| **Входные данные** |
|--------------------|
| 2                  |
| 3                  |
| 5                  |

| **Выходные данные** |
|---------------------|
| 25                  |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
c = int(input())
print((a + b)*c)
```
