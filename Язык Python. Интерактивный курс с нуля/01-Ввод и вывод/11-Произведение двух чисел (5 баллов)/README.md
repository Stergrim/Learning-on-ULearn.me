# Произведение двух чисел

<p align="center">
<b>Произведение двух чисел</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Вам даны два целых числа *A* и *B*, каждое от *0* до *10*. Выведите их произведение.

**Входные данные**

В первой строке дано целое число *A*, во второй – целое число *B*.

**Выходные данные**

Выведите произведение чисел *A* и *B*.

**Пример**

| **Входные данные** |
|--------------------|
| 2                  |
| 2                  |

| **Выходные данные** |
|---------------------|
| 4                   |

| **Входные данные** |
|--------------------|
| 5                  |
| 6                  |

| **Выходные данные** |
|---------------------|
| 30                  |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
print(a*b)
```
