# Ира и томаты

<p align="center">
<b>Ира и томаты</b> <br>
ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>
ввод:
стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

Ира ходила в магазин *N* дней. Треть из этих дней Ира покупала по *A* томатов за раз, из которых *B* были испорчены (возможно все). Во все остальные дни она покупала *C* томатов за раз, из которых *D* были испорчены (возможно все).

Вычислите, сколько всего свежих томатов купила Ира.

**Входные данные**

В первой строке дано целое число *N*, которое лежит в диапазоне от *3* до *99*. Гарантируется, что оно делится на *3*.

Во второй строке – целое число *A* *(1≤A≤100)*.

В третьей – целое число *B* *(0≤B≤A)*.

В четвертой – целое число *C* *(1≤C≤100)*.

В пятой – целое число *D* *(0≤D≤C)*.

**Выходные данные**

Выведите ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 3                  |
| 2                  |
| 0                  |
| 3                  |
| 2                  |

| **Выходные данные** |
|---------------------|
| 4                   |

| **Входные данные** |
|--------------------|
| 3                  |
| 2                  |
| 2                  |
| 3                  |
| 3                  |

| **Выходные данные** |
|---------------------|
| 0                   |


Все тесты пройдены, задача сдана:
```py
n = int(input())
a = int(input())
b = int(input())
c = int(input())
d = int(input())
print((n // 3)*(a - b) + (2*n // 3)*(c - d))
```
