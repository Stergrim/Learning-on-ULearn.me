Витя несёт друзьям *n* конфет, после каждого пройденного километра он съедает *k* из них. Сколько конфет Витя донесёт друзьям, если до них идти ещё *m* километров?

Конфет у Вити много, поэтому можете считать, что они не кончатся до конца пути.

**Входные данные**

В первой строке дано целое число *n*, во второй строке – число *k*, в третьей – *m*.

Каждое число лежит в диапазоне от *1* до *100*.

**Выходные данные**

Выведите ответ на задачу.

**Пример**

| **Входные данные** |
|--------------------|
| 5                  |
| 2                  |
| 2                  |

| **Выходные данные** |
|---------------------|
| 1                   |

| **Входные данные** |
|--------------------|
| 10                 |
| 2                  |
| 5                  |

| **Выходные данные** |
|---------------------|
| 0                   |


Все тесты пройдены, задача сдана:
```py
n = int(input())
k = int(input())
m = int(input())
print(n - m*k)
```
