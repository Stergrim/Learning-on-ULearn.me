<p align="center">

ограничение по времени на тест:
1 секунда <br>
ограничение по памяти на тест:
256 мегабайт <br>

стандартный ввод <br>
вывод:
стандартный вывод <br>
</p>

В этой задаче вам даны два числа — *A* и *B*. Считайте их оба и выведите на экран.

**Входные данные**

В первой строке дано число *A*, во второй — *B*. Оба числа не превосходят *10*.

**Выходные данные**

В первой строке выведите число *A*, во второй — число *B*.

**Пример**

| **Входные данные** |
|--------------------|
| 6                  |
| 4                  |

| **Выходные данные** |
|---------------------|
| 6                   |
| 4                   |


Все тесты пройдены, задача сдана:
```py
a = int(input())
b = int(input())
print(a)
print(b)
```
